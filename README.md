# UZHSWAP Contracts (UniswapV3 Fork)

This Hardhat project can be used to deploy all necessary contracts to the [UZH Ethereum network](http://uzheth.business.uzh.ch)

## Deployment

```
hardhat run scripts/deploy.ts --network uzh
```
